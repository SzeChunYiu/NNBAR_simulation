## Goal A — populate plan 09 (data dictionary)

For every parquet output file produced by the simulation, fill the
per-file column tables in §5 (Carbon), §6 (Silicon), §7 (Beampipe),
§10 (LeadGlass), §11 (PMT), §12 (GPUEnergy) to the depth currently
shown in §8 (TPC).

Each column entry needs:

- `name` (parquet column name)
- `dtype` (int32, int64, float32, float64, string, bool)
- `units` (mm, ns, MeV, electrons, photons, dimensionless)
- `semantics` (one-line description)
- `Class` (A / B / C from plan 01)
- `rule` (which §3 rule of plan 01 governs the classification)
- `produced_by` (SD class or EventAction line range)
- `consumed_by` (downstream functions / plans that read it)

Read sources:

- `NNBAR_Detector/src/output/ParquetOutputManager.cc`
- `NNBAR_Detector/src/sensitive/CarbonSD.cc`
- `NNBAR_Detector/src/sensitive/SiliconSD.cc`
- `NNBAR_Detector/src/sensitive/TubeSD.cc`
- `NNBAR_Detector/src/sensitive/ScintillatorSD.cc`
- `NNBAR_Detector/src/sensitive/LeadGlassSD.cc`
- `NNBAR_Detector/src/sensitive/PMTSD.cc`
- `NNBAR_Detector/src/core/EventAction.cc`
- `NNBAR_Detector/include/hits/NNbarHit.hh`

The TPC pattern in §8 is the template. Match its column-table format
exactly so plan 09 stays uniform.

## Goal B — populate plan 10 (macro inventory)

For every `.mac` under `NNBAR_Detector/macro/**` and
`NNBAR_Detector/macros/**`, add an entry with:

- macro path
- purpose
- key Geant4 commands invoked (focus on `/calibration/...`,
  `/source/...`, `/run/beamOn`, `/particle_generator/...`)
- output files / sample target
- definitive **status**: `active`, `legacy`, or `retired`

Status decision rules:

- `active` — referenced in `reconstruction.md` example commands or in
  a registered sample (plan 03), or thesis-bound study macros under
  `macro/studies/`.
- `legacy` — superseded by an active counterpart (e.g. per-species
  cosmic macros superseded by CRY in plan 21) but still runnable for
  historical reproduction.
- `retired` — broken, redundant, or replaced by a non-macro path
  (e.g. `run_all_calibrations.mac` is replaced by plan 52
  orchestration).
